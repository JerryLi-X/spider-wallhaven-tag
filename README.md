# Spider Wallhaven TAG 特别版

采集 [wallhaven.cc](https://alpha.wallhaven.cc) 上特定的 TAG 图片到程序目录下

# 安装和运行

## Linux 用户

```bash
git clone https://github.com/JerryLi-X/spider_wallhaven_tag.git
cd spider_wallhaven_tag
python3 ./main.py
```

## Windows 用户

系统要求：64 位

下载地址：打 exe 包会出现迷之错误，正在解决
